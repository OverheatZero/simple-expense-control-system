# simple-expense-control-system

Sistema de controle de Despesas

Esse é um projeto desenvolvido em C++(C++17) com o objetivo de praticar conceitos importantes da linguagem, como:
-Programação Orientada a Objetos
-Sintaxe específica da linguagem
-Manipulação de dados

Funcionalidades:

-Manipulação de despesas (CRUD)
-Armazenamento de valores e descrições
-Listagem das despesas
-Ainda está em desenvolvimento, portanto algumas funcionalidades ainda não estão totalmente implementadas.

Tecnologias:

-C++ (padrão 17)

Objetivo do Projeto:

Ter uma maior familiarização com o ambiente e sintaxe da linguagem, juntamente com sua estrutura e boas práticas, além de aprender novos conceitos,
visando evoluir como desenvolvedor. Sugestões e/ou feedbacks são muito bem vindos.

Obrigado!
