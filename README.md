# simple-expense-control-system
<h1>Sistema de controle de Despesas</h1>
Esse é um projeto desenvolvido em C++(C++17) com o objetivo de praticar conceitos importantes da linguagem, como:

- Programação Orientada a Objetos
- Sintaxe específica da linguagem
- Manipulação de dados 

<h3>Funcionalidades:</h3>

- Manipulação de despesas (CRUD)
- Armazenamento de valores e descrições
- Listagem das despesas
- Ainda está em desenvolvimento, portanto algumas funcionalidades ainda não estão totalmente implementadas.
  
<h3>Tecnologias:</h3>

- C++ (padrão 17)
- CMake 3.20+
- Compilador compatível com C++17 (testado com GCC 13.3) 

<h3>Como Executar:</h3>

<h4>Pré-requisitos:</h4>

- CMake 3.20+
- Compilador C++ com suporte a C++17 

<h4>Build:</h4>

```
cmake -S . -B build
cmake --build build
```

<h4>Executar:</h4>

Linux / macOS

```
./build/simple_expense_control_system
```
Windows

```
.\build\simple_expense_control_system.exe
```
<h3>Objetivo do Projeto:</h3> Ter uma maior familiarização com o ambiente e sintaxe da linguagem, juntamente com sua estrutura e boas práticas, além de aprender novos conceitos, visando evoluir como desenvolvedor. Sugestões e/ou feedbacks são muito bem vindos.
